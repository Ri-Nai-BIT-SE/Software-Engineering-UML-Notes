# 需求工程过程
- **过程**是一组相关活动的**集成**，通过这些活动的执行，可以**完成一项任务**或者**达到一个目标**
- 需求工程过程是系统开发中**需求开发活动**的集成，它的模版是产生一个能够在用户环境下解决用户业务问题的系统方案
- 需求工程过程可能会表现出极大的差异，但是除了少数情况之外，**主要的需求工程活动是比较固定的**
